# Fake News Detection Using Machine Learning

## Description
Fake News Detection is a Machine Learning based project that detects whether news is real or fake using both text and images. The system uses NLP and image processing techniques to analyze news content and provide accurate predictions.

---

## Features
- Detects fake and real news
- Text-based news analysis
- Image-based news detection
- Machine Learning prediction
- User-friendly interface

---

## Technologies Used
- Python
- Machine Learning
- NLP
- OpenCV
- Scikit-learn
- Flask
- HTML, CSS, JavaScript

---

## Installation

1. Clone the repository

```bash
git clone https://github.com/NivedithaDesamsetti/Fake_News_Detection_Using_MachineLearning.git
```

2. Open project folder

```bash
cd Fake_News_Detection_Using_MachineLearning
```

3. Install required packages

```bash
pip install -r requirements.txt
```

4. Run the project

```bash
python app.py
```

---

## Output
The system predicts whether the uploaded news is Fake or Real.
